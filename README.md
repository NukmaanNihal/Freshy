# Freshy

Freshy is a clean and modern static website template for an online shop. It includes multiple pages, responsive design, and a polished user interface built with HTML, CSS, and JavaScript.

## Features

- Responsive layout for desktop and mobile
- Static pages for home, shop, cart, checkout, contact, and more
- Interactive JavaScript behavior for shopping interactions
- Clean UI styling using CSS

## Technologies Used

- HTML
- CSS
- JavaScript

## Folder Structure

```
freshy/
├── about.html
├── app.html
├── best-deals.html
├── cart.html
├── checkout.html
├── contact.html
├── index.html
├── shop.html
├── script.js
├── style.css
└── img/
```

## How to Run

1. Open the `freshy` folder in your code editor.
2. Open `index.html` in a web browser.
3. Optionally, use a local development server for a better experience.
